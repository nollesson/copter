# Quadcopter
To hopefully design a reasonably stable quad copter from 'scratch' with an ArduinoOne for the flight controlles.

# Dependencies
* Ubuntu 16
* avr-gcc
* avr-libc
* avrdude
* Bazel
* user in dialout group
